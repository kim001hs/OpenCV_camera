# OpenCV_camera
video recorder using OpenCV

# 기능

### r 키를 누르면 녹화가 시작됩니다 다시 누르면 녹화가 종료됩니다

### 스페이스바를 누르먼 색반전, 윤곽선 모드로 변경이 가능합니다.

### tab 키를를 누르면 화면이 좌우반전됩니다.

### 녹화중엔 빨간 점이 화면에 표시됩니다.

### ESC키를 누르면 종료됩니다.
